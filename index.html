Feed on news

Fang helps you stay up to date with your favorite news websites and blogs.

Currently Fang is in an early beta test. Download a package for your platform if you'd like to help test it.

Changelog

  • New build system with packages for Windows, macOS, and Linux
  • Refreshed UI

  • Option to follow the system's light or dark mode
  • System tray icon support on Windows
  • Context menus for feeds
  • More reliable web UI
  • Support for automatically re-discovering feed URL if it changed on the server
  • Ported to Qt 6
  • Fixed single instance check
  • Numerous bugs and crashes fixed
  • Unit tests and code coverage report added
